Overview

Two Sides of the Same Coin

SVM and SVR share the same mathematical foundation but solve fundamentally different problems.

C

Support Vector Machine

A powerful supervised learning algorithm for classification tasks. SVM finds the optimal hyperplane that separates different classes with maximum margin.

  • Binary and multi-class classification
  • Maximizes margin between classes
  • Outputs discrete class labels
  • Robust to outliers
R

Support Vector Regression

An adaptation of SVM for regression tasks. SVR fits a function within an Œµ-tube, tolerating small errors while minimizing overall deviation.

  • Continuous value prediction
  • Œµ-insensitive loss function
  • Outputs real numbers
  • Sparse solution via support vectors
Mathematical Foundation

The Theory Behind SVM & SVR

Understanding the optimization problems and key mathematical concepts.

SVM Classification

Finding the Optimal Separating Hyperplane

Given training data \(\{(x_i, y_i)\}_{i=1}^{n}\) where \(y_i \in \{-1, +1\}\), SVM finds a hyperplane \(w \cdot x + b = 0\) that separates the classes.

The Optimization Problem

SVM maximizes the margin (distance between hyperplane and nearest points) by solving:

Primal Formulation
$$\min_{w, b} \frac{1}{2} \|w\|^2$$ $$\text{subject to: } y_i(w \cdot x_i + b) \geq 1, \forall i$$

Soft Margin SVM

For non-linearly separable data, we introduce slack variables \(\xi_i\):

Soft Margin Formulation
$$\min_{w, b, \xi} \frac{1}{2} \|w\|^2 + C \sum_{i=1}^{n} \xi_i$$ $$\text{s.t.: } y_i(w \cdot x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0$$
üéØ
The Margin

The margin equals \(\frac{2}{\|w\|}\). Maximizing this is equivalent to minimizing \(\|w\|^2\).

Decision Function

After solving, predictions are made using:

Classification Rule
$$f(x) = \text{sign}\left(\sum_{i \in SV} \alpha_i y_i K(x_i, x) + b\right)$$
‚≠ê
Support Vectors

Only points with \(\alpha_i > 0\) (on or inside the margin) affect the decision boundary. These are the support vectors.

SVR Regression

Fitting Within an Œµ-Tube

Given training data \(\{(x_i, y_i)\}_{i=1}^{n}\) where \(y_i \in \mathbb{R}\), SVR finds a function \(f(x) = w \cdot x + b\) that deviates at most \(\varepsilon\) from targets.

The Œµ-Insensitive Loss

SVR uses a special loss function that ignores errors smaller than Œµ:

Œµ-Insensitive Loss Function
$$L_\varepsilon(y, f(x)) = \begin{cases} 0 & \text{if } |y - f(x)| \leq \varepsilon \\ |y - f(x)| - \varepsilon & \text{otherwise} \end{cases}$$

The Optimization Problem

SVR minimizes model complexity while staying within the Œµ-tube:

SVR Primal Formulation
$$\min_{w, b, \xi, \xi^*} \frac{1}{2} \|w\|^2 + C \sum_{i=1}^{n} (\xi_i + \xi_i^*)$$ $$\text{s.t.: } \begin{cases} y_i - (w \cdot x_i + b) \leq \varepsilon + \xi_i \\ (w \cdot x_i + b) - y_i \leq \varepsilon + \xi_i^* \\ \xi_i, \xi_i^* \geq 0 \end{cases}$$
üìä
The Œµ-Tube

Points inside the tube (error < Œµ) have \(\xi_i = \xi_i^* = 0\) and don't affect the model. Only boundary/outside points matter.

Prediction Function

The regression function depends only on support vectors:

SVR Prediction
$$f(x) = \sum_{i \in SV} (\alpha_i - \alpha_i^*) K(x_i, x) + b$$
üí°
Sparsity

Points inside the Œµ-tube have \(\alpha_i = \alpha_i^* = 0\). The model is defined entirely by points on or outside the tube boundaries.

The Kernel Trick

Handling Non-Linear Data

Kernels implicitly map data to higher dimensions without explicit computation.

Linear Kernel
$$K(x_i, x_j) = x_i \cdot x_j$$

Simplest kernel. Works when data is linearly separable. No transformation applied ‚Äî operates in original feature space.

Polynomial Kernel
$$K(x_i, x_j) = (\gamma \cdot x_i \cdot x_j + r)^d$$

Captures feature interactions up to degree d. Good for problems where combinations of features matter.

RBF (Gaussian) Kernel
$$K(x_i, x_j) = \exp(-\gamma \|x_i - x_j\|^2)$$

Most popular kernel. Maps to infinite-dimensional space. Works well for most non-linear problems.

Sigmoid Kernel
$$K(x_i, x_j) = \tanh(\gamma \cdot x_i \cdot x_j + r)$$

Similar to neural network activation. Can behave like a two-layer perceptron under certain parameters.
